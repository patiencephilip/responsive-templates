## Usage

```bash
git clone https://github.com/yourname/responsive-templates.git
```

Open any template folder and launch `index.html` in your browser. That's it.

To customise:
1. Replace placeholder text with your own content
2. Swap `--accent` color in `:root` to match your brand
3. Update the font imports if you want different typography
4. Drop your project images into the `work-card-img` sections

## Contributing

Want to add a template or improve an existing one? Contributions are very welcome.

1. Fork this repo
2. Create a branch: `git checkout -b template/your-template-name`
3. Add your template in a new folder with a self-contained `index.html`
4. Open a pull request with a short description and screenshot

**Guidelines:**
- Pure HTML, CSS, and vanilla JS only — no frameworks, no npm, no build tools
- Must be fully responsive
- Must include a live demo link in your PR
- One file per template where possible

See [CONTRIBUTING.md](./CONTRIBUTING.md) for full details.
